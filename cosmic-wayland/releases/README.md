# COSMIC Wayland releases

Stable release snapshots for the Magic Mouse COSMIC Wayland package.

## v1.4.3 stable

Path: [`v1.4.3-stable/`](./v1.4.3-stable/)

This snapshot documents the tested v1.4.3 stable package:

- stateful COSMIC workspace switching through `cosmic-ws`
- GUI and CLI control panels
- HID permission repair flow
- scrollable GUI tabs
- desktop/browser direction controls
- Magic Mouse-style SVG icon
