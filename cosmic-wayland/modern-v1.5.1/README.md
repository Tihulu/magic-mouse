# Magic Mouse COSMIC Modern UI v1.5.1

This is the real modern COSMIC control panel, not a Tk theme patch over the old layout.

It keeps the stable v1.4.8 backend and installs a new lightweight Tk control panel with:

- dark Tihuluwave-style cards
- clear mode indicator under the session line
- stable/default workspace and browser direction controls
- sensitivity presets: Default, Tihulu, Sensitive, Relaxed, Very Relaxed, Custom
- custom X threshold, Y threshold, cooldown, and axis lock fields
- quick actions for scroll direction, HID repair, setup verification, CLI, and workspace tests

Install:

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/Tihulu/magic-mouse/main/cosmic-wayland/modern-v1.5.1/install.sh)"
```

Return to the stable simple v1.4.8 UI:

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/Tihulu/magic-mouse/main/cosmic-wayland/stable-v1.4.8/install.sh)"
```
